# Python-
本仓库包含一份完整的 Python 学习笔记（Jupyter Notebook 格式），从零开始讲解 Python 编程的核心概念。每一部分都配有简洁的代码示例和注释，适合初学者快速上手，也适合作为日常编程的参考手册。

无论你是刚接触编程，还是想巩固 Python 基础，这份笔记都能为你提供清晰、实用的指导。

内容概览
笔记共分为 9 个主要章节，循序渐进地覆盖 Python 最常用的知识点：

1. 变量与基本数据类型 – 变量命名、整数、浮点数、字符串基础

2. 字符串操作 – 拼接、大小写转换、空白处理、转义字符

3. 列表、元组与字典 – 增删改查、排序、切片、遍历、嵌套结构

4.条件判断与循环 – if、for、while，以及 break、continue 的用法

5. 函数 – 定义与调用、返回值、默认参数、任意数量的参数

6. 类与面向对象 – 类的定义、实例化、继承、方法重写

7. 文件与异常处理 – 读写文件、处理常见异常（如 FileNotFoundError）

8. 数据可视化 – 使用 matplotlib 绘制折线图、散点图、随机漫步

9. CSV 数据处理 – 读取 CSV 文件、提取数据、绘制温度趋势图

每个章节都配有实际可运行的代码片段，并附有详细注释，方便理解。

章节详情
1. 变量与基本数据类型
介绍如何定义变量、命名规则，以及整型、浮点型、字符串的基本操作。包含类型转换、简单数学运算等实用技巧。

2. 字符串操作
涵盖字符串的拼接、大小写转换（title()、upper()、lower()）、删除空白（rstrip()、lstrip()、strip()）以及转义字符（\t、\n）的使用。

3. 列表、元组与字典
讲解列表的增删改查、排序（sort()、sorted()）、切片、复制；元组的不可变特性；字典的键值对操作、遍历和嵌套。还包含列表解析式等高级用法。

4. 条件判断与循环
详细说明 if-elif-else 逻辑判断，以及 for 和 while 循环的使用，并演示了 break、continue 如何控制循环流程。

5. 函数
学习如何定义和调用函数，理解形参与实参、返回值、默认参数，以及如何接收任意数量的位置参数和关键字参数。

6. 类与面向对象
介绍类的基本结构（__init__、实例属性、方法），并通过继承示例展示如何扩展父类功能。涵盖方法重写和组合等面向对象思想。

7. 文件与异常处理
讲解如何使用 with 语句安全读写文件，以及使用 try-except 捕获和处理常见异常（如除零错误、文件不存在等），使程序更加健壮。

8. 数据可视化
基于 matplotlib 绘制专业的折线图、散点图，并演示了颜色映射、尺寸调整、图表保存等技巧。还实现了一个随机漫步模拟，直观展示数据轨迹。

9. CSV 数据处理
通过处理真实的气温数据集，学习如何解析 CSV 文件、提取日期和温度数据，并绘制高低温区域图，展示数据变化趋势。
